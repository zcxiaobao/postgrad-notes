    # 文章总结

    1. 本文主要是将深度学习的算法应用到能量(负荷)分解中，主要用到了三种算法:
    • 长短期记忆人工神经网络(LSTM)
    • 降噪自动解码器
    • 回归各设备启动时间、结束时间和平均功率需求的网络
    1. 数据来源于实际采集，采集率有两种:
    • 1次/s
    • 1次/6s
    1. 数据来源包括 5 个房间，每个房间包括 5 种电器:
    • kettle
    • firedge
    • washing machine
    • microwave
    • dish washer
    1. 论文所用数据集来自 UK-DALE
    2. 训练数据由两部分组成: 合成综合数据、真实数据，比例是 1:1
    3. 所用激活函数是 linear 和 tanh。
    4. 使用 NILMTK's 提供的 Electric_get_activations() 方法提取激活(???激活是负荷印记吗)
    5. 最后使用 TP FP FN F1 等参数来评价模型的好坏，同时与之前出现的算法(CO FHMM) 进行对比，对比发现，DNN 有更好的价值。

    # 知识学习

    1. RNN: 循环神经网络知识

    2. 文章中提到 RNN 存在梯度爆炸和消失问题，一般有三种方案解决

    • 合理的初始化权重值。初始化权重，使每个神经元尽可能不要取极大值或极小值，躲避开梯度消失的区域
    • 使用 relu(线性整流函数) 代替 sigmoid 和 tanh 作为激活函数
    • 使用其他结构的 RNN，例如长短时记忆网络 LTSM 和 Gated Recurrent Unit（GRU）
    1. 宏平均值、召回率、精准率等
